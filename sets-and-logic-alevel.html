เซตคืออะไร?

เซต (Set) คือ กลุ่มของสิ่งของที่กำหนดได้ชัดเจน ว่าอะไรอยู่ในกลุ่มหรือไม่อยู่ในกลุ่ม

นิยาม
เซต คือ การรวมกลุ่มของสิ่งต่าง ๆ ที่มีลักษณะเฉพาะร่วมกัน เรียกสมาชิกแต่ละตัวว่า สมาชิก (Element)
ตัวอย่าง
A = {1, 2, 3, 4, 5} → เซตของจำนวนนับตั้งแต่ 1 ถึง 5
B = {a, b, c} → เซตของพยัญชนะภาษาอังกฤษสามตัวแรก
การเขียนแทนเซต

แบบแจกแจงสมาชิก

เขียนระบุสมาชิกทุกตัวลงในวงเล็บปีกกา

A = {2, 4, 6, 8}

แบบบอกเงื่อนไข

เขียนเงื่อนไขที่สมาชิกต้องเป็นไปตามนั้น พร้อมระบุเอกภพสัมพัทธ์

A = {x ∈ ℕ | x เป็นจำนวนคู่ และ x ≤ 8}
💡 เทคนิค A-Level — เขียนเงื่อนไขให้ชัด
ควรระบุเอกภพสัมพัทธ์ด้วยเสมอ เช่น x ∈ ℕ หรือ x ∈ ℤ เพื่อให้ความหมายของเซตชัดเจน ไม่คลาดเคลื่อน
🔢
สมาชิกและจำนวนสมาชิก

ใช้สัญลักษณ์ ∈ แทน "เป็นสมาชิกของ" และ ∉ แทน "ไม่เป็นสมาชิกของ"

ถ้า A = {1, 2, 3} แล้ว
2 ∈ A (2 เป็นสมาชิกของ A)
5 ∉ A (5 ไม่เป็นสมาชิกของ A)
จำนวนสมาชิก n(A)
n(A) คือจำนวนสมาชิกของเซต A
เช่น A = {a, b, c, d} → n(A) = 4
⭐
เซตพิเศษที่ต้องรู้
∅
เซตว่าง
เซตที่ไม่มีสมาชิก n(∅) = 0
U
เอกภพสัมพัทธ์
เซตที่ครอบคลุมสมาชิกทุกตัวในบริบทนั้น ๆ
⊂
เซตย่อย (สับเซต)
A ⊂ B: ทุกสมาชิกของ A อยู่ใน B
P(A)
เพาเวอร์เซต (Power Set)
n(P(A)) = 2ⁿ เมื่อ n = n(A)
ตัวอย่าง เพาเวอร์เซต
A = {1, 2} → P(A) = {∅, {1}, {2}, {1,2}}
n(A) = 2 → n(P(A)) = 2² = 4
⚠️ ข้อควรระวัง
เซตว่าง ∅ เป็นสับเซตของทุกเซต เสมอ
∅ ⊂ A สำหรับทุก A เสมอ
📋
เซตเท่ากัน และการตรวจสมาชิกที่ซับซ้อน

เซตเท่ากัน (A = B)

มีสมาชิกเหมือนกันทุกตัว ลำดับไม่สำคัญ

{1,2,3} = {3,1,2} ✓

ระวัง: ปีกกาซ้อน!

A = {∅, 1, {1}} มีสมาชิก 3 ตัว
สมาชิกคือ: ∅, 1, และ {1}
{1} ∈ A ✓ แต่ {1} ⊂ A ✗ (เพราะ {1} เป็นสมาชิก ไม่ใช่สับเซต)

💡 สับเซตและสมาชิก — กับดักข้อสอบ
กำหนด A = {∅, 1, {1}}
• ∅ ∈ A (จริง — ∅ เป็นสมาชิกของ A)
• ∅ ⊂ A (จริง — ∅ เป็นสับเซตของทุกเซต)
• {1} ∈ A (จริง — {1} เป็นสมาชิกของ A)
• {1} ⊂ A (จริง — เพราะสมาชิกของ {1} คือ 1 ซึ่งอยู่ใน A ด้วย)
• {{1}} ⊂ A (จริง — เพราะ {1} เป็นสมาชิกของ A)
• 1 ∈ {1} (จริง) แต่ 1 ∈ A ก็จริงเช่นกัน
⚠️ สูตรจำนวนสับเซต
ถ้า n(A) = n แล้ว A มีสับเซตทั้งหมด 2ⁿ เซต
และ A มีสับเซตที่แท้จริง (ไม่รวมตัวเอง) ทั้งหมด 2ⁿ - 1 เซต
หัวข้อ 2 · การดำเนินการบนเซต
∪
ยูเนียน (Union) A ∪ B
นิยาม
A ∪ B = เซตของสมาชิกที่อยู่ใน A หรือ B (หรือทั้งสอง)
ตัวอย่าง
A = {1, 2, 3, 4}, B = {3, 4, 5, 6}
A ∪ B = {1, 2, 3, 4, 5, 6}
💡 เทคนิคจำ
∪ มาจาก Union = "รวมกัน" — เอาสมาชิกทุกตัวจากทั้งสองเซต (ไม่ซ้ำกัน)
∩
อินเตอร์เซกชัน (Intersection) A ∩ B
นิยาม
A ∩ B = เซตของสมาชิกที่อยู่ใน A และ B พร้อมกัน
ตัวอย่าง
A = {1, 2, 3, 4}, B = {3, 4, 5, 6}
A ∩ B = {3, 4}

ถ้า A ∩ B = ∅ เรียกว่า A และ B เป็น เซตไม่มีสมาชิกร่วมกัน

A'
คอมพลีเมนต์ (Complement) A'
นิยาม
A' = เซตของสมาชิกใน U ที่ ไม่อยู่ ใน A
ตัวอย่าง
U = {1,2,3,4,5,6,7,8}, A = {1,2,3}
A' = {4,5,6,7,8}
สูตรสำคัญ
n(A') = n(U) − n(A)
A ∪ A' = U
A ∩ A' = ∅
−
ผลต่าง (Difference) A − B
นิยาม
A − B = เซตของสมาชิกที่อยู่ใน A แต่ไม่อยู่ ใน B
A − B = A ∩ B'
ตัวอย่าง
A = {1,2,3,4}, B = {3,4,5}
A − B = {1,2}
📐
สูตรจำนวนสมาชิก (สำคัญมาก!)
n(A ∪ B) = n(A) + n(B) − n(A ∩ B)
n(A ∪ B ∪ C) = n(A) + n(B) + n(C)
− n(A ∩ B) − n(B ∩ C) − n(A ∩ C)
+ n(A ∩ B ∩ C)
ตัวอย่างโจทย์
นักเรียน 30 คน เรียนคณิต 18 คน เรียนวิทย์ 20 คน เรียนทั้งสอง 8 คน
จงหาจำนวนนักเรียนที่เรียนอย่างน้อยหนึ่งวิชา

วิธีทำ: n(M ∪ S) = 18 + 20 − 8 = 30 คน
🔁
สมบัติและกฎสำคัญของเซต

กฎการสลับที่ (Commutative)

A ∪ B = B ∪ A
A ∩ B = B ∩ A

กฎการเปลี่ยนหมู่ (Associative)

(A ∪ B) ∪ C = A ∪ (B ∪ C)
(A ∩ B) ∩ C = A ∩ (B ∩ C)

กฎการกระจาย (Distributive)

A ∩ (B ∪ C) = (A ∩ B) ∪ (A ∩ C)
A ∪ (B ∩ C) = (A ∪ B) ∩ (A ∪ C)

กฎเอกลักษณ์ (Identity)

A ∪ ∅ = A
A ∩ U = A
🔄
กฎของเดอมอร์แกน (De Morgan's Laws)
กฎของเดอมอร์แกน — ต้องจำให้ได้!
กฎของเดอมอร์แกนอธิบายความสัมพันธ์ระหว่าง คอมพลีเมนต์, ยูเนียน และอินเตอร์เซกชัน
กฎที่ 1:   (A ∪ B)' = A' ∩ B'
กฎที่ 2:   (A ∩ B)' = A' ∪ B'
💡 วิธีจำ
เมื่อนำคอมพลีเมนต์เข้าไป ให้ กระจายเครื่องหมาย ' เข้าทุกตัว แล้วเปลี่ยน ∪ ↔ ∩
เช่น (A ∪ B)' → A' และ B' แล้วเปลี่ยน ∪ เป็น ∩ ได้ A' ∩ B'
ตัวอย่างที่ 1 — ตรวจสอบกฎที่ 1
กำหนด U = {1,2,3,4,5,6}, A = {1,2,3}, B = {2,3,4}

วิธีที่ 1 (ด้านซ้าย):
A ∪ B = {1,2,3,4} → (A ∪ B)' = {5,6}

วิธีที่ 2 (ด้านขวา):
A' = {4,5,6}, B' = {1,5,6}
A' ∩ B' = {5,6} ✓ (ได้เหมือนกัน)
⚠️ ประยุกต์ใช้กฎเดอมอร์แกน
กฎนี้ใช้บ่อยในการหาค่า n(A'∩B') หรือ n(A'∪B'):
n(A'∩B') = n((A∪B)') = n(U) − n(A∪B)
n(A'∪B') = n((A∩B)') = n(U) − n(A∩B)
หัวข้อ 3 · แผนภาพเวนน์-ออยเลอร์
⭕
แผนภาพเวนน์สองเซต

ใช้วงกลมแทนเซต แต่ละบริเวณมีความหมายต่างกัน

บริเวณต่าง ๆ

  • A เท่านั้น = A − B
  • B เท่านั้น = B − A
  • ส่วนกลาง = A ∩ B
  • ทั้งหมด = A ∪ B
  • นอกวง = (A ∪ B)'

สูตรคำนวณ

n(A only) = n(A) − n(A∩B)
n(B only) = n(B) − n(A∩B)
n(neither) = n(U) − n(A∪B)
⭕
แผนภาพเวนน์สามเซต
โจทย์ประยุกต์
นักเรียน 50 คน สำรวจงานอดิเรก: วาดรูป (A) 25 คน, ดนตรี (B) 20 คน, กีฬา (C) 22 คน
A∩B = 8, B∩C = 7, A∩C = 6, A∩B∩C = 3

n(A∪B∪C) = 25+20+22−8−7−6+3 = 49 คน
ไม่มีงานอดิเรก = 50−49 = 1 คน
📐
การหาค่าสูงสุด–ต่ำสุด (Max-Min) ของ n(A∩B)
หลักการ
เมื่อทราบ n(U), n(A), n(B) แต่ไม่ทราบ n(A∩B) สามารถหาขอบเขตได้ดังนี้

ค่าต่ำสุด (Minimum): n(A ∩ B) ≥ max(0, n(A)+n(B)-n(U))
ค่าสูงสุด (Maximum): n(A ∩ B) ≤ min(n(A), n(B))
ตัวอย่าง
n(U) = 30, n(A) = 20, n(B) = 18

ค่าต่ำสุด: max(0, 20+18-30) = max(0, 8) = 8
ค่าสูงสุด: min(20, 18) = 18

ดังนั้น 8 ≤ n(A ∩ B) ≤ 18
💡 แนวคิดภาพ
ค่าต่ำสุด เกิดเมื่อซ้อนทับกันน้อยที่สุด = ยืดวงออกจากกันเท่าที่ U จะอนุญาต
ค่าสูงสุด เกิดเมื่อเซตเล็กกว่าถูกซ้อนเต็มอยู่ในเซตใหญ่กว่า (เซตเล็ก ⊂ เซตใหญ่)
โจทย์แนว A-Level — สามเซต
โรงเรียนมีนักเรียน 100 คน สำรวจชมรม A (60 คน), B (50 คน), C (40 คน)
ถ้าไม่ทราบส่วนซ้อนใด ๆ จงหาค่ามากสุดและน้อยสุดของ n(A∩B∩C)

ค่ามากสุด: min(n(A), n(B), n(C)) = min(60,50,40) = 40 คน (เป็นไปได้เมื่อ C ⊂ B ⊂ A)
ค่าน้อยสุด: max(0, n(A)+n(B)+n(C)-2n(U)) = max(0, 150-200) = 0 คน
* ค่าน้อยสุดสำหรับสามเซตใช้สูตร max(0, n(A)+n(B)+n(C)−2n(U))
บทที่ 2 · ตรรกศาสตร์ (Logic)
💬
ประพจน์ (Proposition)
นิยาม
ประพจน์ คือ ประโยคบอกเล่าหรือปฏิเสธที่ สามารถบอกได้ว่าจริงหรือเท็จ เพียงอย่างเดียว

✓ เป็นประพจน์

  • "2 + 2 = 4" (จริง)
  • "กรุงเทพฯ อยู่ในจีน" (เท็จ)
  • "ดวงอาทิตย์ขึ้นทางทิศตะวันออก"

✗ ไม่เป็นประพจน์

  • "x + 5 = 10" (ไม่รู้ค่า x)
  • "สวัสดีครับ" (คำทักทาย)
  • "ปิดประตูด้วย" (คำสั่ง)
💡 เทคนิคดูว่าเป็นประพจน์ไหม
ลองถามตัวเองว่า "ประโยคนี้จริงหรือเท็จ?" ถ้าตอบได้ = ประพจน์ ถ้าตอบไม่ได้ = ไม่เป็นประพจน์
🔢
ค่าความจริง (Truth Value)

ประพจน์ทุกตัวมีค่าความจริงเพียงหนึ่งค่า คือ จริง (T) หรือ เท็จ (F)

p = "5 เป็นจำนวนคี่" → ค่าความจริง = T (จริง)
q = "4 หารด้วย 3 ลงตัว" → ค่าความจริง = F (เท็จ)

ประพจน์ที่ใช้ตัวแปร เช่น p, q, r เรียกว่า ตัวแปรประพจน์

สัจนิรันดร์ (Tautology)

ประพจน์ที่จริงทุกกรณี ไม่ว่าค่าความจริงของตัวแปรจะเป็นอะไร

p ∨ ~p
(ทุกแถวจริง)

ข้อขัดแย้ง (Contradiction)

ประพจน์ที่เป็นเท็จทุกกรณี ไม่ว่าตัวแปรจะมีค่าความจริงใด

p ∧ ~p
(ทุกแถวเท็จ)

ตัวอย่างที่ 1 — นิเสธของ p → q
~(p → q) ≡ p ∧ ~ q
ภาษาไทย: "ถ้าฝนตก แล้วถนนเปียก"

ตัวอย่างที่ 2 — นิเสธของ p ∧ q
~(p ∧ q) ≡ ~ p ∨ ~ q
ภาษาไทย: "ฝนตก และ อากาศเย็น"
นิเสธ: "ฝนไม่ตก หรือ อากาศไม่เย็น" ✓
💡 เทคนิค A-Level — ตรวจสอบสัจนิรันดร์เร็ว
วิธี "สมมติให้เป็นเท็จ": ถ้าสมมติให้ประพจน์เป็นเท็จแล้วเกิดข้อขัดแย้ง → เป็นสัจนิรันดร์
สำหรับ A → B: สมมติ A = T, B = F แล้วดูว่าเกิดขัดแย้งหรือไม่
ตัวอย่าง — ตรวจสอบ (p ∧ q) → (~ p → r) เป็นสัจนิรันดร์หรือไม่
สมมติให้เป็นเท็จ: หน้า (p ∧ q) = T, หลัง (~ p → r) = F
จากหลัง = F: ~ p = T (p = F) และ r = F
จากหน้า = T: p = T และ q = T
เกิดขัดแย้ง (p = F และ p = T) → เป็นสัจนิรันดร์ ✓
หัวข้อ 8 · ประโยคเปิดและตัวบ่งปริมาณ (∀, ∃)
∀
ประโยคเปิด (Open Sentence) และตัวบ่งปริมาณ
นิยาม — ประโยคเปิด
ประโยคที่มีตัวแปร ยังตัดสินจริง/เท็จไม่ได้จนกว่าจะกำหนดค่าตัวแปร
เช่น "x + 3 > 5" หรือ "x² = 4" เป็นประโยคเปิด

∀ ตัวบ่งปริมาณ "ทุกตัว"

"∀x P(x)" อ่านว่า "สำหรับ x ทุกตัว, P(x) จริง"
จะเป็น เท็จ ถ้ามี x แม้แต่ตัวเดียว ทำให้ P(x) เท็จ

∀ x ∈ ℝ: x² ≥ 0
จริง ✓

∃ ตัวบ่งปริมาณ "บางตัว"

"∃x P(x)" อ่านว่า "มี x บางตัว ที่ P(x) จริง"
จะเป็น จริง ถ้ามี x อย่างน้อยหนึ่งตัว ทำให้ P(x) จริง

∃ x ∈ ℝ: x + 5 = 3
จริง (x = −2) ✓
ตัวอย่างในเอกภพสัมพัทธ์ต่าง ๆ
เอกภพ ℕ (จำนวนนับ 1, 2, 3, …):
• ∀ x ∈ ℕ: x > 0 → จริง (จำนวนนับทุกตัว > 0)
• ∃ x ∈ ℕ: x² = 4 → จริง (x = 2)

เอกภพ ℤ (จำนวนเต็ม …, −1, 0, 1, …):
• ∀ x ∈ ℤ: x > 0 → เท็จ (x = −1 ทำให้เท็จ)
• ∃ x ∈ ℤ: x² = 2 → เท็จ (ไม่มีจำนวนเต็มที่ยกกำลัง 2 ได้ 2)

เอกภพ ℝ (จำนวนจริง):
• ∀ x ∈ ℝ: x² ≥ 0 → จริง
• ∃ x ∈ ℝ: x² = 2 → จริง (x = √2)
~
นิเสธของประพจน์ที่มีตัวบ่งปริมาณ
กฎสำคัญ — ต้องจำ!
~(∀ x P(x)) ≡ ∃ x ~ P(x)
~(∃ x P(x)) ≡ ∀ x ~ P(x)

เมื่อนิเสธ: ∀ เปลี่ยนเป็น ∃ และ ∃ เปลี่ยนเป็น ∀ แล้วนิเสธ P(x)
ตัวอย่าง — นิเสธของประพจน์ ∀ และ ∃
1. ประพจน์: "นักเรียนทุกคนสอบผ่าน" (∀x: สอบผ่าน)
นิเสธ: "มีนักเรียนบางคนสอบไม่ผ่าน" (∃x: ~สอบผ่าน) ✓

2. ประพจน์: ∃ x ∈ ℝ: x > 100
นิเสธ: ∀ x ∈ ℝ: x ≤ 100 → เท็จ ✓

3. ประพจน์: ∀ x ∈ ℤ: x² ≥ 0
นิเสธ: ∃ x ∈ ℤ: x² < 0 → เท็จ (เพราะประพจน์ต้นจริง)
💡 เทคนิค A-Level — นิเสธประพจน์ผสม
โจทย์มักให้ประพจน์รูป "ถ้า P แล้ว Q" ที่ P หรือ Q มีตัวบ่งปริมาณ
เช่น "ถ้านักเรียนทุกคนสอบผ่าน แล้วมีนักเรียนบางคนได้รับรางวัล"
→ P = ∀x[สอบผ่าน], Q = ∃x[ได้รับรางวัล]
→ ~(P→Q) ≡ P ∧ ~Q = ∀x[สอบผ่าน] ∧ ∀x[ไม่ได้รับรางวัล]
∀∃
ตัวบ่งปริมาณสองตัวแปร
ลำดับสำคัญมาก!
∀ x ∃ y P(x,y) ≠ ∃ y ∀ x P(x,y)
การสลับลำดับของ ∀ และ ∃ อาจเปลี่ยนค่าความจริงได้
ตัวอย่าง — เอกภพ ℝ
1. ∀ x ∈ ℝ, ∃ y ∈ ℝ: y > x
→ "สำหรับจำนวนจริงทุกตัว มีจำนวนจริงบางตัวที่มากกว่า" → จริง ✓

2. ∃ y ∈ ℝ, ∀ x ∈ ℝ: y > x
→ "มีจำนวนจริงบางตัวที่มากกว่าจำนวนจริงทุกตัว" → เท็จ ✗

3. ∀ x ∈ ℕ, ∃ y ∈ ℕ: x + y = 10
เอกภพ ℕ: x=1→y=9 ✓, x=2→y=8 ✓, … x=9→y=1 ✓, แต่ x=10→y=0 ✗ (0 ∉ ℕ) → เท็จ
เอกภพ ℤ: มีคำตอบเสมอ → จริง
⚠️ นิเสธของตัวบ่งปริมาณสองตัวแปร
~(∀ x ∃ y P(x,y)) ≡ ∃ x ∀ y ~ P(x,y)
~(∃ x ∀ y P(x,y)) ≡ ∀ x ∃ y ~ P(x,y)

หลักการ: ย้อนตัวบ่งปริมาณทุกตัว (∀↔∃) แล้วนิเสธ P ในที่สุด
